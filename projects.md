
# Beginner Rust Projects for Python Developers

Start with small projects.

Do not start with a huge backend system immediately.

## Project 1: CLI Todo App

Concepts:

- structs
- Vec
- file handling
- Result
- ownership

## Project 2: File Word Counter

Concepts:

- reading files
- strings
- error handling
- iterators

## Project 3: JSON Parser with Serde

Concepts:

- structs
- serialization
- deserialization
- Result

## Project 4: Simple HTTP API

Concepts:

- Axum
