# Rust for Python Developers Bundle

This repository is free.

I also created a paid Rust for Python Developers Bundle for people who want a more structured learning path.

The bundle is designed for developers who know Python or web development and want to learn Rust step by step.

It covers:

- Rust basics
- ownership
- borrowing
- memory model
- structs
- enums
- traits
- error handling
- Python vs Rust mental model
- practical projects

You can find it here:

https://tobiweissmann.gumroad.com/l/wcylm

You do not need to buy the bundle to use this free roadmap.
