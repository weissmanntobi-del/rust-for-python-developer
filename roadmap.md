# Rust for Python Developers Roadmap

## Phase 1: Understand What Python Hides

Before learning Rust deeply, understand what Python hides from you:

- memory allocation
- references
- object identity
- garbage collection
- runtime type checking
- dynamic dispatch

## Phase 2: Learn Memory Basics

Learn:

- CPU
- RAM
- stack
- heap
- pointer
- reference
- allocation
- deallocation
- copy
- move

## Phase 3: Learn Rust Ownership

Understand:

- every value has one owner
- values are moved by default
- ownership prevents use-after-free bugs
- ownership removes the need for a garbage collector

## Phase 4: Learn Borrowing

Understand:

- immutable borrowing
- mutable borrowing
- references
- the rule: many readers or one writer

## Phase 5: Learn Rust Data Modeling

Learn:

- structs
- enums
- pattern matching
- traits
- impl blocks

## Phase 6: Build Small Projects

Start with:

- CLI todo app
- file reader
- CSV parser
- JSON API client
- mini HTTP server
- log analyzer
